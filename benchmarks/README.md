# Benchmarks

This directory contains benchmark runners, task banks, and generated results.

## Scope
1. Determinism and telemetry runs.
2. Quant sweep diagnostics.
3. Task-bank based execution reports.

## Canonical Operations Docs
1. Quant sweep operations: `docs/QUANT_SWEEP_RUNBOOK.md`
2. General operations: `docs/RUNBOOK.md`
3. Testing policy: `docs/TESTING_POLICY.md`

## Directory Layout
1. `benchmarks/task_bank/`: task definitions.
2. `benchmarks/configs/`: matrix/profile configs.
3. `benchmarks/results/`: generated artifacts.

## Minimal Commands
Run a quant sweep via the operational runbook command set. Do not duplicate command matrices in this file.
