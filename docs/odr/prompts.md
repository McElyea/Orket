# ODR Prompt Templates (v1)

## Architect Output Shape
Required sections:
1. `### REQUIREMENT`
2. `### CHANGELOG`
3. `### ASSUMPTIONS`
4. `### OPEN_QUESTIONS`

## Auditor Output Shape
Required sections:
1. `### CRITIQUE`
2. `### PATCHES`
3. `### EDGE_CASES`
4. `### TEST_GAPS`

ODR v1 parses only these sections and ignores model orchestration concerns.
