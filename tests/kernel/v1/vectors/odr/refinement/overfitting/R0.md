# Requirement Spec

## Scope
Define offline assistant memory behavior.

## Definitions
- Offline mode: no network access.

## Non-goals
- No social feed integrations.

## Invariants
- Memory runs without external APIs.

## Failure Codes
- REQ_SCOPE_DRIFT

## Acceptance Tests
- Verify no outbound network call in offline mode.
