# Requirement Spec

## Scope
Define deletion policy for local audit records.

## Definitions
- Retained delete: remove after retention window.

## Non-goals
- No remote archival.

## Invariants
- Local-only storage.
- Single deletion policy is enforced.

## Failure Codes
- REQ_POLICY_CONTRADICTION

## Acceptance Tests
- Verify deletion behavior is deterministic.
- Verify 30-day retention before deletion.
